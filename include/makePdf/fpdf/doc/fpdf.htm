FPDF

FPDF([string orientation [, string unit [, mixed format]]])

バージョン

1.0

説明

FPDFクラスのコンストラクタ。 ページの方向(orientation)、 すべてのメンバ関数で利用する寸法の種類(unit)※フォントサイズは除く、 ページの用紙サイズ(format)、 を指定することができます。

パラメータ

orientation
デフォルトのページの方向を指定できます。 指定可能な値（完全一致する必要があります）：
  • P または Portrait
  • L または Landscape
（何も入力しなかった場合の）デフォルトの値はPです。 訳者注：Portraitは縦長、Landscapeは横長を意味します。
unit
デフォルトの寸法の種類を指定できます。 指定可能な値：
  • pt: ポイント(point)
  • mm: ミリ(millimeter)
  • cm: センチ(centimeter)
  • in: インチ(inch)
ポイントは1/72インチと同じで、およそ0.35mm(1インチ=2.54cm換算)です。 ポイントは印刷物の世界では一般的な寸法です。 ※ちなみにフォントサイズはこの寸法(ポイント)で表現されます

（何も入力しなかった場合の）デフォルトの値はmmです。
format
ページの用紙サイズです。 指定可能な値（完全一致する必要があります）：
  • A3
  • A4
  • A5
  • Letter
  • Legal
または、カスタムした用紙サイズを指定できます。 幅(width)と高さ(height)の2つの要素をもつ配列をパラメータに渡します。 ※この幅と高さの単位は、寸法の種類(unit)で指定したものとして扱われます
