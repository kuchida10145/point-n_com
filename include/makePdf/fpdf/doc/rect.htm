Rect

Rect(float x, float y, float w, float h [, string style])

バージョン

1.0

説明

長方形を出力します。 （境界線を）描画するだけ、塗りつぶすだけ（境界線なし）、または描画と塗りつぶしの両方を指定できます。

パラメータ

x
左上の角のx座標。
y
左上の角のy座標。 Ordinate of upper-left corner.
w
幅(width)。
h
高さ(height)。
style
描画スタイルを指定します。 指定できる値：
  • D または空: (境界線を)描画します。 デフォルト値。
  • F: 塗りつぶし。
  • DF または FD: (境界線の)描画と塗りつぶしの両方。

参照

SetLineWidth(), SetDrawColor(), SetFillColor().
