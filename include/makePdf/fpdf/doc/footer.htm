Footer

Footer()

バージョン

1.0

説明

ページフッタを出力するために利用されます。 AddPage()、Close()メンバ関数により自動的にコールされますので、明示的にコールしてはいけません。 FPDFクラスではデフォルトでは空になっていますので、 独自の処理を行いたい場合は、FPDFクラスを継承し、このメンバ関数をオーバライドしてください。

例

class PDF extends FPDF
{
function Footer()
{
    //ページの下端から1.5 cm上に縦座標をセットする
    $this->SetY(-15);
    //Arial italic 8のフォントを指定する
    $this->SetFont('Arial','I',8);
    //ページ番号をセンタリングして出力
    $this->Cell(0,10,'Page '.$this->PageNo(),0,0,'C');
}
}
